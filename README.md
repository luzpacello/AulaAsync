# AulaAsync

App para gestión académica personal.

## 🧱 Stack

- Backend: Node + TypeScript + Prisma
- DB: Supabase (PostgreSQL)
- Web: Vue + Vite
- Mobile: React Native (Expo)

## 🚀 Cómo correr

### Backend
cd backend
npm install
npm run dev

### Frontend Web
cd frontend-web
npm install
npm run dev
